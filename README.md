# WeBank 课程 —— 第一小组

### 项目介绍
**MusicChain--音链**

### 项目目录结构

- `day1 - day*` 存放每天每个成员的作业，并在周报中使用
- `MusicChain` 存放项目部分文件，主体代码参照下方链接
- `周报` 文件夹存放周报内容
  - `assets` 图片资源文件夹
  - `Week*.md` 周报内容
 - `实训课程报告` 存放每个人的实训总结报告
